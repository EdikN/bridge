# Одноклассники (VK Bridge)

Документация по интеграции Одноклассников через VK Bridge SDK.

## Описание
В последнее время Одноклассники поддерживают запуск мини-игр через VK Bridge SDK. В этом режиме игра инициализируется стандартным вызовом `VKWebAppInit`, но параметры платформы и социальные функции (вступление в группу и др.) отличаются от ВКонтакте.

В Playgama Bridge для этого реализован специальный класс `OkVkPlatformBridge`, который наследует всю логику VK Bridge, но корректирует её под Одноклассники.

---

## Конфигурация
Для корректной работы вступления в сообщество на обеих платформах, необходимо настроить ID групп в файле `playgama-bridge-config.json`.

```json
{
  "social": {
    "joinCommunity": {
      "vk": "213542253",
      "ok": "70000048656390"
    }
  }
}
```

*   **vk**: Числовой ID группы ВКонтакте (не короткое имя).
*   **ok**: Числовой ID группы Одноклассников (берется из ссылки `ok.ru/group/123456...`).

---

## Определение платформы
Платформа определяется автоматически при инициализации. SDK ищет следующие параметры в URL:
1.  `vk_client=ok`
2.  `vk_ok_app_id` (присутствует при запуске из ОК)

Если эти параметры найдены, SDK устанавливает `platformId` в значение `ok`. В консоли при инициализации появится сообщение:
`[Bridge] Platform detected: ok`

---

## Особенности и ограничения
1.  **Лидерборды**: На текущий момент не поддерживаются для Одноклассников через VK Bridge. Метод `bridge.leaderboards.isSupported` вернет `false`, а тип будет `not_available`.
2.  **Добавление в закладки**: Отключено для Одноклассников (`isAddToFavoritesSupported = false`).
3.  **Вступление в группу**: Метод `joinCommunity` автоматически перенаправляет пользователя на `https://ok.ru/group/ID` после вызова метода подписки в SDK.

---

## Использование в коде
Код игры остается универсальным:

```js
// Вступить в сообщество (автоматически подставит нужный ID из конфига)
bridge.social.joinCommunity()
    .then(() => console.log('Успех'))
    .catch(err => console.error(err))
```
